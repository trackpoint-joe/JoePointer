Help & Instructions
Job Search Command Center
What is this tool?

The Job Search Command Center is your private, browser-based tracker for managing job applications and sending personalized portfolio links to employers — all without a backend or login.

Your data lives entirely in your browser's localStorage. Nothing is sent to a server. Use the JSON backup/restore feature to move data between devices.

📋 Applications Tab

Your master list of applications. Each row shows company, role, status badge, applied date, and follow-up date.

  • 🔗 Copy URL — Copies your personalized portfolio link to clipboard
  • 👁 Preview — Opens your portfolio with the company's industry template applied
  • 📧 Draft Email — Opens your mail client with subject, welcome message, and portfolio URL pre-filled
  • 🗑 Delete — Permanently removes the application (shows a confirmation prompt first)

Use the filter buttons to narrow the list. Click any row to edit the full record.

⚠️ Rows with overdue follow-up dates display a red warning flag. Check the Dashboard tab to see all overdue items at a glance.
📊 Dashboard Tab

A visual summary of your search health — stats cards, pipeline breakdown pie chart, industry chart, and overdue follow-up alerts. Click any pie slice or legend item to filter the Applications tab.

➕ New Application

Opens in Quick Add mode. Click "+ More Details" to expand dates, salary, contacts, notes, and ATS analysis fields.

  • Industry Template — Sets the color scheme on the employer's personalized link
  • Welcome Message — Personal note shown at the top of the portfolio page
  • Job Description — Paste and click Analyze Keywords to see resume match. Green = in your resume; yellow = gaps to address
💡 The "Applied Date" defaults to today. Set a follow-up reminder for 7–10 days after applying.
🎤 Interview Prep

Appears on any application in Phone Screen, Technical, or Final Round stage. Uses the Claude API (your key) to generate 8 questions to ask + 8 questions to prepare, all personalized to the company, role, and stage. Story Bank matches suggest which career story to tell for each prep question.

💾 Backup & Restore

Your data is stored in this browser on this device only.

  • ↓ CSV — Spreadsheet-compatible export (Excel / Google Sheets)
  • ↓ JSON — Full backup. Use to move between devices or restore after clearing storage
  • ↑ Import — Restores from a JSON backup. Merges with existing data
🔒 Recommended: export a JSON backup at the end of each week.
💡 Pro Tips
  • Click 🎭 Samples to load realistic sample data and preview all dashboard features
  • Click any pie chart slice in the Dashboard to instantly filter by that status or industry
  • Press Escape to close any open panel or modal
  • The 🔗 URL generates a link your employer opens directly — no login required on their end
